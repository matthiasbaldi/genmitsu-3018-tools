## Index

-   [USB over IP (USBIP)](./usbip/README.md)

## Equipment

The scripts and guides written here in the Github repository or in the linked blog above are tested with the following equipment:

-   Raspberry Pi 3/4 Model B with Raspberry Pi OS Lite
-   Windows PC as client
-   [Candle GRBL Controller](https://github.com/Denvi/Candle)
-   [LightBurn Laser Software](https://lightburnsoftware.com/)
